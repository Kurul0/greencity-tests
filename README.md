# Тестування сторінки подій GreenCity

Цей репозиторій містить практичну реалізацію процесу ручного тестування вебзастосунку GreenCity.

### Короткий опис проєкту
**Мета проєкту:** Проведення тестування відкритого функціоналу сторінки подій (Events) та розробка структурованої тестової документації у вигляді тест-кейсів. 

У рамках проєкту було проаналізовано такі ключові аспекти системи:
- Робота навігації (бургер-меню) на нестандартних розмірах екрана.
- Логіка авторизації та розмежування прав доступу (спроба взаємодії з контентом без реєстрації).
- Валідація форм вводу (введення даних кирилицею у поле для вводу пошти).

**Об'єкт тестування:** Сторінка [GreenCity Events](https://www.greencity.cx.ua/#/greenCity/events)
